Welcome To ImagineMemory's Progress Tracker!

On this page you can track the statistics of the games that you have played!

    The stats we track are:

  • The number of games you have played.
  • The number of games you have won.
  • The number of successful moves you have made.
  • The number of unsuccessful moves you have made.
  • Lastly the time taken to complete a game.

The best of these statistics are presented to you so you can see how well you are doing!
